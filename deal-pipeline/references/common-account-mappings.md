# Common Account Mappings — Canadian SMB

## Revenue
| Source Label | Standard Category | Note |
|---|---|---|
| Revenue / Sales / Billings / Fees earned | Gross Revenue | |
| Service revenue / Contract revenue | Gross Revenue | |
| Equipment rental / rental income | Gross Revenue (core ops) or Other Income (incidental) | Investigate |
| Fluid hauling / freight revenue | Gross Revenue | |
| Returns / allowances / discounts | Less: Returns & Allowances | |
| Gain on sale of assets / disposal of equipment | NON-RECURRING — exclude from EBITDA | Flag |
| Interest income | Non-operating — below EBIT | |
| CEWS / CEBA / government subsidies | NON-RECURRING — exclude | Flag |
| Insurance proceeds (non-operating) | NON-RECURRING | Flag |
| Other income / misc income | Other Income — investigate if recurring | Flag |

## Cost of Goods Sold
| Source Label | Standard Category |
|---|---|
| Driver wages / field wages | Direct Labour |
| Fuel / diesel / lubricants | Fuel |
| Subcontractors / contract labour | Subcontractors |
| Vehicle maintenance / truck repairs | Equipment Operating Costs |
| Equipment repairs & maintenance | Equipment Operating Costs |
| Materials / supplies / consumables | Materials & Supplies |
| Depreciation in COGS | Pull out for EBITDA calculation |

## Operating Expenses
| Source Label | Standard Category | Flag? |
|---|---|---|
| Owner salary / shareholder salary | Compensation — Owner/Shareholder | Always |
| Owner draw / drawings | Compensation — Owner/Shareholder | Always |
| Family member wages | Compensation — Owner/Shareholder | Always — verify role |
| Management fees to holdco | Professional Fees | Always — often add-back |
| Consulting fees (to owner-related) | Professional Fees | Flag — may be disguised comp |
| Admin salaries / office wages | Compensation — Management & Admin | |
| Rent / lease — premises | Rent & Occupancy | Flag if related-party landlord |
| Accounting / bookkeeping / audit | Professional Fees | |
| Legal fees | Professional Fees | Flag if abnormally high |
| Insurance | Insurance | |
| Owner vehicle / personal vehicle | Vehicle — flag personal use | Always |
| Cell phone — owner personal | Other OpEx | Partial add-back |
| Travel & entertainment | Travel & Entertainment | Flag if high % of revenue |
| Advertising / marketing | Advertising & Marketing | |
| Utilities / telephone | Telephone & Utilities | |
| Bad debt / doubtful accounts | Bad Debt Expense | Flag if one-time |
| Depreciation & amortization | D&A — separate for EBITDA | |
| Interest expense / bank charges | Interest Expense — below EBIT | |
| Donations / charitable | Other OpEx | Add-back |

## Balance Sheet — Assets
| Source Label | Standard Category | Flag? |
|---|---|---|
| Bank / chequing / savings | Cash & Cash Equivalents | |
| Accounts receivable / trade AR | Accounts Receivable | Review aging |
| GST/HST receivable | Other Current Assets | |
| Inventory / parts / stock | Inventory | |
| Prepaid / deposits | Prepaid Expenses & Deposits | |
| Due from shareholder / related party | Due from Related Parties | Always — not collectible post-close |
| Vehicles / fleet (cost) | PPE — Vehicles | |
| Equipment / machinery (cost) | PPE — Equipment | |
| Buildings / leasehold improvements | PPE — Property | |
| Accumulated depreciation | Less: Accumulated Depreciation | |

## Balance Sheet — Liabilities
| Source Label | Standard Category | Flag? |
|---|---|---|
| Accounts payable / trade payable | Accounts Payable | |
| Accrued liabilities | Accrued Liabilities | |
| Current portion LTD | Current Portion of Long-Term Debt | |
| Income tax payable | Income Taxes Payable | |
| GST/HST payable | Other Current Liabilities | |
| Due to shareholder / shareholder loan | Due to Related Parties | Flag — structure matters |
| Term loan / bank loan | Long-Term Debt | Note lender, rate, maturity |
| Equipment financing / lease obligations | Long-Term Debt | |
| Deferred revenue | Other Current Liabilities | Flag — quality of revenue |
